Homelab.

Welcome to my homelab! This is my personal playground where I experiment with self-hosting, system administration, networking, and software development. It's constantly evolving, but here is a snapshot of my current setup.

⚙️ The Hardware Fleet

Node 1: Homser (Primary Application Server)

  • OS: Ubuntu 24.04.4 LTS (via CasaOS)
  • CPU: Intel Core i3-1005G1 @ 1.20GHz (4 Cores)
  • RAM: 8 GB DDR4
  • Role: Core application, media streaming, and smart home automation hub.

Node 2: Pi Hole (Network Security)

  • OS: Debian Linux
  • CPU: Intel Atom N455 @ 1.66GHz (2 Cores)
  • RAM: 2 GB
  • Role: Network-wide ad-blocking and local DNS management.

Planned Nodes

  • Raspberry Pi: Dedicated monitoring server.
  • Dedicated NAS Storage: Multi-HDD array optimized for high-capacity photo and data backups.

🐳 Deployed Services

I run bare-metal OS installations utilizing CasaOS as my primary management layer, allowing me to easily deploy and maintain containerized services via Docker.

Management & Dashboards

  • CasaOS: Cloud-management interface layer.
  • Homepage: Sleek, unified landing page asset.
  • Glances: Real-time system monitoring.
  • Nginx: Reverse proxy handling traffic routing.

Media & Automation

  • Jellyfin: Open-source media streaming server.
  • Navidrome: High-performance music streaming server.
  • Sonarr / Radarr / Prowlarr: Automated media acquisition stack.

Smart Home Hub

  • Home Assistant: Centralized local smart home automation engine.
  • ESPHome: Custom firmware management for ESP8266/ESP32 microcontrollers.

Cloud & Utilities

  • Nextcloud: Private productivity cloud and initial file storage.
  • Watch Your LAN: Network security scanner for tracking connected devices.
  • UpSnap: Wake-on-LAN dashboard to remotely wake sleeping machines.
  • Dockhand: Automated Docker container backup and maintenance tool.

🖧 Network Architecture

  • Mesh VPN: My network is built around Tailscale. This allows me to seamlessly and securely access all my services (like CasaOS, Jellyfin, and Home Assistant) from anywhere in the world without exposing any ports to the public internet.
